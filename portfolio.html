PuppyLove (10/2021 - 12/2021)

GitHub

PuppyLove is an online application that allows college students to match their dogs with other local dogs to set up safe playdates. PuppyLove was the most technically challenging project I have done up to this point, but the experience I gained leaning React and Node.js will be valuble additions to my diversifying skillset. I also got more leadership experience as I was the manager of the front-end, meaning that I would approve all changes to the code and would be in charge of making the overall design.

Aggie Registration Alert (8/2020 - 12/2020)

GitHub

Aggie Registration Alert is a program designed to alert students whenever a previously full class gains an opening. Since I was a full stack developer, I collaborated with the back-end team to pull class and section data from the Texas A&M server using Python and co-led the front-end team to design the website and user interface using HTML, CSS, and JavaScript.
